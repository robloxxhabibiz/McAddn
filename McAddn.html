Upstash Redis
Upstash Database
Hubungkan Upstash supaya data sync ke semua browser
Cara setup Upstash (2 menit, gratis)
  1. Buka upstash.com → klik Sign Up → login pakai Google
  2. Klik Create Database → isi nama bebas → pilih region Asia Pacific (Singapore) → Create
  3. Scroll ke bawah ke bagian REST API
  4. Copy UPSTASH_REDIS_REST_URL (tombol copy di sampingnya)
  5. Copy UPSTASH_REDIS_REST_TOKEN
  6. Paste ke 2 kolom di bawah → Simpan
Gratis · 10.000 request/hari · Data permanen
Keduanya ada di halaman database Upstash bagian REST API

Groq API Key
Masukkan key untuk aktifkan fitur AI
Key disimpan di browser kamu saja, tidak dikirim ke server manapun selain Groq.
Cara dapat key Groq (gratis)
  1. Buka console.groq.com
  2. Daftar / login
  3. Klik API Keys → Create API Key
  4. Copy & paste ke sini
Tier gratis: 14.400 request/hari · Tidak perlu kartu kredit
